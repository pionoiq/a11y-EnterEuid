<br/><br/>

**<h3>프로젝트 목표 ✨ </h3>**
  1. 협업을 통해 소통하는 법 배우기<br>
  2. 접근성을 준수하는 마크업 하기<br>
  3. 검색엔진 최적화 해보기
   
   
   <br /><br />
